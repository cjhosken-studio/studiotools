**Studiotools** is an open-source **USD** based pipeline tool for **VFX and CG Productions**.  

---

## Building

You will need to build OpenUSD and install it into usd/install. Specifically, the usdview and python modules are needed.


## Requirements
Blender 5.0+
Houdini 21.0+
Nuke 17.0+
